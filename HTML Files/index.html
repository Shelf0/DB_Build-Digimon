Digimon Data Project



For my project, I decided to build a database of Digimon. My hopes for this project were that I would be able to build a database taht could tell the user what moves were available to a particular Digimon based on its Attribute. Each digimon has its own Attribute and there are about 10 total Attributes.

I was inspired to do this after finding a data set on Kaggle.com that had these files on Digimon. So, much like the ETL project, I took my data from the source, tranformed it, in order to better use it and then loaded it into a SQL Database. However, I started to run into some issues. I wanted to start using SQLalchemy with this database, but it seemed I had to have a SQLite database in order to do that and I couldn't manage to convert my .sql file into a .sqlite file properly. So, I decided it would be best to move forward and put a simple website together that could provide some information on Digimon.
